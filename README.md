![Resume 1](images/resumes/Resume_Sept29-Jan26.png?raw=true)
This was the resume used to apply to the mentioned 30 internship positions. It was also used for some applications during the Fall semester. I went to multiple resume workshops to best understand how to format a resume and how to properly convey my role and impact on my projects. One takeaway from those was that numbers and measured metrics helps sell why the project matters and stands out to recruiters.

Strengths:
- Technologies are bolded
- Each project has metrics that helped emphaize skill expressed on the project

Weaknesses:
- Skills section is at the bottom when it should ideally be at the top to be seen first
- Some metrics for projects are not incredibly relevant to the develompent process and don't always express the skills learned (event booking application emphasizes speed when it should emphazise OOP and how they were applied)
